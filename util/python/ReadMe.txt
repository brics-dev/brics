1. Install Python
---------------------------------------------------------------------------------------------------

1.1. Download install for version 3.5+ from www.python.org/downloads/

1.2. Install in your default or preferred system location

1.3. Modify your PATH environment variable to include the directory where python.exe is

2. How to install Python plugins
---------------------------------------------------------------------------------------------------
Generally, this is done executing "python -m pip install ${PLUGIN}"

2.1. psycopg2 is the Postgres drive for Python. To install, run:

        python -m pip install psycopg2
